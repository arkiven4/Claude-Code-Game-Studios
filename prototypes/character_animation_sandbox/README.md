# Character and Animation Sandbox

A prototype for testing character models and animations from the `imported_assets/` directory.

## Features

- **Model Selection**: Switch between different character GLB/GLTF models.
- **Animation Playback**: List and play animations associated with the character or external animation libraries.
- **Camera Controls**: Orbit, zoom, and pan to inspect models from all angles.
- **Visual Reference**: Includes a ground plane and lighting for clear visibility.

## Usage

1. Open `CharacterAnimationSandbox.tscn` in the Godot Editor.
2. Run the scene (F6).
3. Use the UI to select a model and an animation.
4. Mouse controls:
    - **Right Click + Drag**: Orbit camera.
    - **Middle Mouse + Drag**: Pan camera.
    - **Scroll**: Zoom in/out.
