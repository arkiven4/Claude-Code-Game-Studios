# QWEN.local.md Template

Copy this file to the project root as `QWEN.local.md` for personal overrides.
This file is gitignored and will not be committed.

```markdown
# Personal Preferences

## Model Preferences
- Configure your preferred model parameters here

## Workflow Preferences
- Always run tests after code changes
- Use /clear between unrelated tasks
- Configure your compacting preferences here

## Local Environment
- Python command: python (or py / python3)
- Shell configuration
- IDE preferences

## Communication Style
- Keep responses concise
- Show file paths in all code references
- Explain architectural decisions briefly

## Personal Shortcuts
- Configure your custom shortcuts here
```

## Setup

1. Copy this template to your project root: `cp .qwen/docs/QWEN-local-template.md QWEN.local.md`
2. Edit to match your preferences
3. Verify `QWEN.local.md` is in `.gitignore` (Qwen Code reads it from the project root)
